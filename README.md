# Instagram-OSINT

A Node.js OSINT tool to extract publicly available information from Instagram profiles.

## Installation
```bash
git clone https://github.com/kyletechie/Insta-OSINT.git &&
cd Insta-OSINT &&
npm install 
```

## Usage 
```bash
npm start <username>
```

### Example:
```bash
npm start instagram
```
